How these numbers work

The model runs 10,000 bracket simulations of the full tournament, using match probabilities from the Dixon-Coles model — the same model that generates the per-match predictions above. After each completed result, the model is refit and the simulations rerun.

Win % is how often each team wins the tournament across those 10,000 runs.

Fair odds = 1/p − 1. If Argentina has a 20% chance, fair decimal odds are 4.0:1 — a £10 bet at fair value returns £50 total (£40 profit). Bookmaker odds will be lower due to their margin.

These are model probabilities, not betting advice.
